Node.JS Stack

Sobre


Como um runtime JavaScript assíncrono orientado a eventos, o Node.js foi projetado para construir aplicativos de rede escalonáveis. Node.js leva o modelo de loop de eventos um pouco mais longe. Ele apresenta um loop de eventos como uma construção de tempo de execução em vez de uma biblioteca. Inibindo a chamada start-the-event-loop. O runtime simplesmente entra no loop de eventos após executar o script de entrada e sai quando nao ha mais callbacks para executar
